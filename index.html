Arcade Game Demo

What is this?

This game is my 5th project for the Udacity's Front-End Web Dev nanodgree. The goal is to create a classic arcage frogger game using the newly acquired skills in Object Oriented JS and HTML5 Canvas. Images and a game loop engine are provided. The rest should be built!

How to Play

Use the arrow keys to move the player across the board. To win you need to reach water, avoiding the bugs.
